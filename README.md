## What is it?
- Proxy server for a Rotten Tomatoes clone group project.
- It connects three service modules: Scoreboard, Tickets & Showtimes, and Audience Reviews.

## Demo
[movie showtimes demo](https://youtu.be/vk07sRRm01M)

## Getting Started
### Installing
- ```npm install```
### Start Local Server
- ```npm start```

## Tech Choices
Client
- Bootstrap
- CSS
	
Server
- Node
- Express
